## 풀이

단순히 Hello World! 를 출력하면 된다

```php
<? echo "Hello World!";
```

```cpp
#include <iostream>
using namespace std;

int main() {
    cout << "Hello World!";
    return 0;
}
```

```javascript
console.log('Hello World!');
```
